# Mercado Insights

## Ideia do App

Plataforma de análise financeira quantitativa para o mercado brasileiro, sem dependência de IA paga. O produto prioriza dados de mercado, indicadores públicos, regras transparentes e cálculos auditáveis.

## Core Features

- Market Overview Dashboard: cotações, volume, variação diária, market cap e histórico comparativo.
- Quantitative Signals: score de oportunidade por momentum, liquidez, valuation, dividendos e risco.
- Portfolio Analysis: cálculo de valor atual, alocação, concentração por ativo, setor e tipo de ativo.
- Fixed Income Simulator: simulação com taxa manual, Selic, CDI e IPCA do Banco Central.
- News Triage: leitura local por palavras-chave positivas e negativas.
- Customizable Alerts: alertas locais de preço com validação contra cotação atual.
- Backtesting Engine: estratégias fixas de IFR, médias móveis e rompimento executadas sobre histórico real.

## Data Providers

- Brapi: ações, ETFs, FIIs, cotações, histórico, fundamentos e dividendos quando disponíveis.
- Banco Central do Brasil: séries temporais públicas via SGS.
- CVM: base futura para fundos, FIIs e documentos públicos.
- Provedores opcionais futuros: Alpha Vantage ou Twelve Data para dados internacionais e indicadores técnicos.

## Style Guidelines

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Light grey (#F0F2F5) for a clean, professional look.
- Accent color: Teal (#008080) for interactive elements and call-to-action buttons.
- Body font: PT Sans.
- Headline font: Space Grotesk.
- Interface orientada a dashboard: densa, clara e objetiva.
