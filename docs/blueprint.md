# **App Name**: Mercado Insights

## Core Features:

- Market Overview Dashboard: Displays key market metrics like intraday quotes, volume, RSI, and moving averages for stocks and ETFs.
- Algorithmic Signals: Generates buy/sell signals based on momentum, mean reversion, and volatility indicators. A composite score provides a simplified overview.
- AI-Powered News Summarizer: Leverages a tool using a Large Language Model (LLM) to summarize financial news articles and classify sentiment, highlighting key insights for users.
- Explainable Recommendations: Suggests 'Top Opportunities' based on the user's risk profile, investment horizon, and preferred sectors, with explanations of the factors influencing each pick.
- Customizable Alerts: Sends email, WhatsApp, or push notifications when specific triggers occur, such as moving average crossovers or price breakouts.
- Backtesting Engine (Basic): Allows users to test simple trading strategies against historical data and view performance metrics like CAGR and Sharpe Ratio.
- User Preference Center: Enables users to define their risk profile, investment goals, preferred sectors, and other criteria to personalize the analysis and recommendations.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Light grey (#F0F2F5) for a clean, professional look.
- Accent color: Teal (#008080) for interactive elements and call-to-action buttons.
- Body font: 'PT Sans' a humanist sans-serif for readability and a modern touch
- Headline font: 'Space Grotesk' a sans-serif font for headings and titles.
- Use a set of consistent, minimalist icons to represent market concepts and categories.
- Incorporate subtle animations to provide feedback on user interactions and data updates.
